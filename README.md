# Receipt as Emotional Ledger

An Anti-Extractive Archive.

A civic infrastructure for transforming disposable consumer receipts into collective emotional and economic knowledge.

This is not a product. This is a reassignment of function.

## Arte Util Project

An Arte Util project by Jess A. Rivera.

## The Fields We Challenge

**Retail Data Analytics** — Redirecting consumer data from corporate extraction to public civic knowledge.

**Consumer Finance** — Making survival spending visible before it enters the extractive financial pipeline.

**The Emotional Labor Economy** — Recording the uncompensated labor of household financial management.

## Structure

- `index.html` — Main archive: urgency, why contribute, submission, archive, emotional frequency, sustainability
- `gallery.html` — Receipt photo gallery
- `style.css` — Visual design (IBM Plex Mono, three-color palette, Hammersley-inspired glyph canvas)
- `app.js` — Submission logic, localStorage persistence, emotional analysis, glyph rendering

## Deploy

This is a static site.

```
git clone https://github.com/jess-vera/r-el.git
cd r-el
vercel
```

Or connect the repo to Vercel for automatic deploys.

## Data

Currently uses browser localStorage for the prototype. Submissions persist per browser. Seed data is included to demonstrate the archive. Future versions will connect to an open, community-governed database.

## Open Source

All code is publicly available. Fork, adapt, and deploy for your community. No proprietary tools, no locked infrastructure.
